# Prism · 棱镜

一款开源的数据可视化大屏项目，帮助你从 0 到 1 学会自己动手制作数据大屏。

## 项目简介

**Prism（棱镜）** 寓意透过数据折射出多维度的洞察。本项目提供完整的从设计到实现的数据大屏搭建流程，覆盖数据接入、图表绘制、布局编排、交互设计等核心环节，适合前端开发者、数据分析师以及所有对数据可视化感兴趣的学习者。

## 项目定位

- 🧩 **学习友好** — 从零开始，循序渐进，每个阶段都有可运行的大屏产物
- ⚙️ **工程规范** — 遵循真实项目开发流程，Git 工作流、模块划分、代码规范一应俱全
- 📊 **实战驱动** — 围绕真实场景的数据进行可视化设计，学完即用
- 🔌 **可扩展** — 模块化架构设计，方便替换数据源、切换图表库或集成到现有系统
